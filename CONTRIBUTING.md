# Contributing

Corrections, independent proof checks, alternative derivations, and formalization work are welcome.

A useful contribution should clearly identify which layer it changes:

- mathematical theorem or proof;
- proof audit;
- executable arithmetic check;
- documentation;
- formalization.

Please keep exploratory claims separate from proved statements. If a proof step is challenged, include either a concrete counterexample to that step or a replacement argument with explicit assumptions.

Before submitting changes, run:

```bash
python run_checks.py
```
